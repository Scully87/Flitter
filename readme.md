  - JQuery
  - PostgresQL
  - BCrypt
  - Rack-Flash
  - Capybara
  - Devise
  - Omiauth
  - Heroku

How to use
----------
Clone the repository:
```shell
$ git clone git@github.com:scully87/flitter.git
```

Change into the directory:
```shell
$ cd flitter
```

Open in command line:
```shell
$ rackup
in browser http://localhost:9292
```
